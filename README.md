# -Emile-Gall-1846-1904-
Fue uno de los pioneros del modernismo y fundó la Escuela de Nancy, de trascendental influencia en la expansión de este movimiento artístico. Fue, asimismo, el creador de tratamientos innovadores en el trabajo del vidrio, para lo cual desarrolló diversas técnicas. Inició su carrera en la fábrica de Scherer & Co., productora de las vidrieras Burgun.
